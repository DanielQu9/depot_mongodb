# 倉庫管理系統
基於 python 以及 mondgdb 寫出來的倉庫管理工具 <br>
版本使用 python@3.12 測試 <br>
依賴項目請使用 <code>"pip install -r requirements.txt"</code> <br>
使用 <code>python gui.py</code> 執行<br>
